# Redirect Rules Draft (2026-07-17)

Two layers: **row-level 301s** generated by `wp si:redirects` (from `_legacy_url` metas written
during transform/promote), and the **pattern rules** below. Final `redirects.csv` (Redirection-
plugin import format: `source,target,regex,code`) is emitted at P5; rows for retired/moved URLs
with real inlinks get hand-crafted upgrades after the Screaming Frog crawl (S1: crawl = sole
redirect source of truth; C14: rank by inlink count + a one-off free-tier backlink lookup).

## 1. What never redirects (preserved verbatim — the whole strategy)

| URL class | Why safe |
|---|---|
| `/blog/YYYY/MM/DD/slug/` posts (EN + `/de/` mirrors) | Articles never change type or slug (06 §3); coverage/video/conference promotions are the exceptions → row-level 301s |
| `/media/{slug}/` presentations | `si_presentation` rewrite base IS `/media/` (06 change #5) — zero redirects for 333 public EN items |
| Root pages that stay pages | untouched |

## 2. Pattern rules (regex; low risk — none of these are in the XML sitemap)

| Source pattern | Target | Note |
|---|---|---|
| `^/blog/category/.*` | `/blog/` | 176 category archives retire (05 §10); hand-craft specific rules ONLY for archives the crawl shows inlinks for (expect ≤ a dozen) |
| `^/de/blog/category/.*` | `/de/blog/` | DE mirror |
| `^/(de/)?blog/tag/.*` | `/blog/` | 55 dead tags |
| `^/category/.*` | `/blog/` | legacy debris (01 §A3: currently 301s to random posts / 404s) |
| `^/recent-news/?$` | `/blog/` | ALM page replaced by real blog index |
| `^/coverage/?$` | *(none)* | KEEP the path — the `si_coverage` archive takes over `/coverage/` (same URL, new engine) |
| `^/international-conferences/?$` | `/conferences/` | index page → CPT archive |

## 3. Row-level 301 sources (generated, joined against the crawl at P5)

1. **Coverage promotions** (~154 posts, R5.1): `/blog/…/slug/` → `/coverage/slug/`.
2. **Video promotions** (webcast/series posts, R3): `/blog/…/slug/` → `/videos/slug/`.
3. **Conference promotions** (`action=promote` rows in conference-map.csv): old post/page URL → `/conferences/slug/`. (`action=create` rows keep the old post as an Article — no redirect.)
4. **Retired pages** (38 junk + review outcomes): → nearest hub or `/` (case-by-case column in classification.csv notes).
5. **DE per-talk pages** (R4.1): `/de/{slug}/` → `/de/media/{new-slug}/`.
